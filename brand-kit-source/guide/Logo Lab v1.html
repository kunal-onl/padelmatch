Each cell subdivided into a sub-grid so the canvas can actually render letters. Default state: scattered "matching" pixels in lime + blue. Animation: pixels cascade and resolve into P (lime) → M (blue) → dissolve back. Heavier feel, suited for splash screens / loading states.
