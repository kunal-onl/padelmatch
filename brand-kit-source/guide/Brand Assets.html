PRIMARY MARK
DEFAULT IDENTITY · 3 BACKGROUNDS
The court-grid mark with TL·lime and BR·blue diagonal — the default static identity. Use this for app icons, favicons, OG cards, hero placements.
03
LETTER MARKS — P & M
LINE-COLORED LETTERFORMS
P (lime) and M (blue) drawn from the court grid itself. Use as decorative endcaps in copy ("…matched on PadelMatch"), as section dividers, in marketing collateral, or as standalone monograms.
04
PATTERN VARIATIONS
STATIC FILL COMPOSITIONS
Alternate cell-fill compositions for editorial / illustrative use — rating result tiles, loading splash, social shareables. Not interchangeable with the primary mark; these reinforce the brand without replacing it.

Self-contained animated SVG using SMIL — no JS required. Drops into any modern browser. Use for "finding your match" loaders, web hero panels, email signatures, video stingers. Download the file, or paste the embed snippet inline.
